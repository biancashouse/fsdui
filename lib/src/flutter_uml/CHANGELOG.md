## 0.1.0

* Initial release.
* `UmlDiagram.plantuml` — renders PlantUML diagrams via a Kroki server.
* `UmlDiagram.mermaid` — renders Mermaid diagrams via a Kroki server.
* Platform-specific rendering: native `<img>` element on web, `SvgPicture.network` on mobile/desktop.
* Configurable Kroki base URL, loading builder, and error builder.
